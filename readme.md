Morning2Do
==========

WeirdMeetup의 ReactNative의 스터디 과제로 진행하였습니다.

## 프로젝트의 배경

사람은 매일 고정적으로 해야할 중요한 일이 있습니다.
매일 고정되고 중요한 일을 하면서도 잊어버리는 일이 다반사 입니다.
그런 중요성을 띈 할 일을 위한 TodoApp 입니다.

## 기능
* TodoItem의 등록, 삭제, 변경
* 매일 등록된 시간에 TodoItem들의 완료여부를 초기화.
